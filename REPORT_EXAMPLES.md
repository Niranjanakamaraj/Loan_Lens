## When Reports are Generated

Reports are generated **exactly at the moment you click Download**, containing:

1. The specific form data you submitted
2. The exact AI predictions from that submission
3. The exact positive/negative factors identified
4. Current date and time stamp
5. All relevant model analysis

Each download creates a new file with timestamp to prevent overwriting.

## Report Data Source

Each report contains data directly from:

- **Form Data**: User's input values (income, loan amount, credit history, etc.)
- **API Response**: XGBoost model's predictions (probability, decision, factors)
- **Timestamp**: When the report was generated

This ensures **complete accuracy** and **audit trail** for each analysis.
